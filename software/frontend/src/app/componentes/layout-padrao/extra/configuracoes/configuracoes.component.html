Bloqueio de rota durante PAD

Aviso O botão abaixo serve para desabilitar a entrada de usuário que não sejam SUPER durante uma realização de viagem do PAD.

Isto é feito para evitar que algum paciente seja cadastrado/alterado durante a viagem, podendo ocasionar em um erro de sincronização de dados ao final do retorno da mesma.
