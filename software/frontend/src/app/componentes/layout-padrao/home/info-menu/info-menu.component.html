Instruções de uso
Paciente cirurgia

Esta aba é dedicada ao cadastro de informações relacionadas aos pacientes encaminhados a cirurgia. Através do submenu é possível:

  • Cadastrar novos pacientes
  • Adicionar nova lesão para pacientes cadastrados
  • Cadastrar histopatológico para as lesões já cadastradas
  • Editar um paciente já cadastrado
  • Filtrar informações dos pacientes e lesões cadastradas
  • Visualizar todas as informações de um paciente cadastrado
Painel de lesões

Este módulo apresenta uma galeria com todas as lesões e suas imagens (se houver) dos pacientescadastrados no sistema.

É uma ótima opção para visualizar as lesões que estão no sistema e apurar a qualidade das mesmas.

Análise de dados

Nesta área será apresentada toda área de análise de dados dos pacientes e lesões cadastradas no sistema.

Você encontrará gráficos, tabelas e conclusões em relação aos dados. Essa funcionalidade aida está em desenvolvimento e será disponibilizada assim que estiver pronta.

Usuário

Este módulo, como o próprio nome sugere, possui as funcionalidades relacionadas aos usuários do sistema. Através dela é possível:

  • Caso você seja um ADMIN, é possível adicionar novos usuários no sistema. Neste caso, esses usuário já estarão aptos acessar o sistema.
  • Visualizar todos os usuários do sistema. Caso seja um ADMIN, você poderá conceder privilégios e acesso aos demais usuários.
  • Alterar os seus dados, como nome de usuário, email e senha.
